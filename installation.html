Installing OpenPNM¶

OpenPNM requires the Scipy Stack (Numpy, Scipy, Matplotlib, etc), which is most conveniently obtained by installing the Anaconda Distribution, so first download and install this if you don’t have it already.

OpenPNM can be installed from the Python Package Index using:

pip install openpnm

Or the source code can be downloaded from Github and installed by running:

The advantage to installing from the source code is that you can edit the files and have access to your changes each time you import OpenPNM.

Finally, OpenPNM is now available as a conda package, so you can also do:

conda install -c conda-forge openpnm
